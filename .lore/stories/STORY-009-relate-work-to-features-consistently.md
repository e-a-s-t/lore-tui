# STORY-009 - Relate work to Features consistently

## User Story

As a Lore user,  
I want Features to be normal artifacts,  
so that Requirements, ADRs, Stories and Tests can relate to Features using the same model as all other artifact relationships.

## Acceptance Criteria

- I can create a Feature.
- I can link Requirements, ADRs, Stories and Tests to the Feature.
- I can inspect the Feature and see its related artifacts.
- I can inspect a Requirement and see which Feature it belongs to.
