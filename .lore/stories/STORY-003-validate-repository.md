# STORY-003 - Validation Repository

## User Story

### Story 1

As a user, I want to run

```shell
lore-cli validate
```

so that I can detect problems in my .lore repository before committing changes.

### Story 2

As a developer, I want to run

```shell
devbox run run-cli validate
```

so that I can detect problems in my .lore repository while developing committing changes.
