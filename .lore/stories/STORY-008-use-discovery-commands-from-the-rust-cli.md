# STORY-008 - Use discovery commands from the Rust CLI

## User Story

As a Lore user, I want to use discovery commands from the Rust CLI so that I can inspect artifacts, search memory, and review traceability without leaving the terminal.

## Acceptance Criteria

- [ ] I can inspect a specific artifact with `lore show`.
- [ ] I can search by text with `lore search`.
- [ ] I can review traceability with `lore trace`.
- [ ] I can review missing links with `lore gaps`.
