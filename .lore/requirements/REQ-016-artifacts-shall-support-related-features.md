# REQ-016 - Artifacts shall support related_features

## Requirement

All Lore artifact types shall support a related_features field.

## Acceptance Criteria:

- Requirements may link to Features using related_features.
- Stories may link to Features using related_features.
- ADRs may link to Features using related_features.
- Tests may link to Features using related_features.
- Features may link to other Features using related_features.
