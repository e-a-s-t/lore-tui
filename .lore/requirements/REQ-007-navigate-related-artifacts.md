# REQ-007 - Navigate related artifacts

## Requirement

Users shall be able to navigate artifacts displayed in the Related pane.

## Acceptance Criteria

- Focus can move between panes.
- Requirements, Stories, ADRs and Tests are selectable.
- Enter opens the selected artifact.
- Back navigation returns to the previous artifact.
