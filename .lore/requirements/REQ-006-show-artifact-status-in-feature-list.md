# REQ-006 - Show artifact status in feature list

## Requirement

The Features pane shall display the status of each Feature.

## Acceptance Criteria

- Status is visible beside the Feature ID.
- Accepted, Draft and Proposed states are distinguishable.
- The active Feature remains highlighted.
