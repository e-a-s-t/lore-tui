# TEST-006 - Related pane supports navigation

## Test Case

Verify that users can move focus to the Related pane and navigate related artifacts.

## Expected Result

- Focus can switch between the Features pane and Related pane.
- Up and Down move the selected related artifact.
- Enter opens the selected related artifact.
- Back returns to the previous artifact.
