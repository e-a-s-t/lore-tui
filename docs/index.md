# Lore

Lore is a Git-native software planning system for humans and AI.  

It helps teams capture requirements, decisions, stories, tests and implementation context as structured Markdown artifacts.  

The goal is not to maximize context.

The goal is to minimize the amount of context required to complete a task successfully.

## Core Principles

- Git-native
- Human and AI friendly
- Plain Markdown
- Traceable
- Language agnostic

## Artifact Model

```text
Idea
  ↓
Feature
 ├── Requirements
 ├── Stories
 ├── ADRs
 └── Tests
        ↓
Implementation
        ↓
Validation
```
